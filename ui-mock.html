Secure Token 付与ツール

端末上のユーザから付与元・付与対象を選び、Secure Token を付与します。

親アカウント(付与元 / Secure Token 保有者)
この親アカウントは標準ユーザです。付与時に管理者アカウント(macadmin)の権限で一時的に管理者へ昇格し、完了後に標準ユーザへ戻します。
子アカウント(付与対象 / Secure Token 未保有)
実行ログ
状態:

Avalonia (.NET 9 / FluentTheme) アプリのUIプレビュー。「付与」押下後、まず パスワードを事前検証し、誤りがあれば昇格に入らず中止します。全て正しければ macadmin 権限で昇格 → 付与 → 降格を自動実行します。
